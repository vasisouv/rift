# Rift

**Turn-based space roguelite. Every run ends in death — but death makes you stronger.**

## Gameplay

Pick a gun, fight your way through sequential encounters, and face a boss at the end of each level. Spend energy on perks and shop items between levels. When you die, your shards carry over to the Void Shop where you buy permanent upgrades that persist across runs.

## Features

- 3 guns with unique mechanics (Plasma Rifle, Shotgun, Sniper)
- Sequential encounters + boss fight per level
- Perks and in-run shop for build customization
- Meta-progression Void Shop with permanent upgrades
- Enemy telegraphing — react to CHARGE, SHIELD, REGEN, STUN
- Guard action to halve incoming damage
- Burn status from crits, Stun from Power Strike
- Keyboard shortcuts: `A` Attack · `S` Power Strike · `D` Guard · `Tab` Cycle Target

## Run Locally

```bash
npm install
npm run dev
```

## Stack

- Vue 3
- Pinia
- Tailwind CSS
- Vite
